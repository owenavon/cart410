Markup.

How to integrate accessibility into the markup of the code.

Tip 1

Headings establish the hierarchy of the content of the page. Therefore it is important to use heading tags in order from h1 tp h4. Screen readers uses headings tags to read content in order from top to bottom.

  • h1

  • h2

  • h3

  • h4

Tip 2

Review the WAI-ARIA Roles. There are 6 categories of ARIA roles:

  • Document structure roles
  • Widget roles
  • Landmark roles
  • Live region roles
  • Window roles
  • Abstract roles
